Ya esta hecho el parser que lee un string donde hay escrito un termino en 
Prolog que cumple es_progresion y lo transforma a un dato del tipo 
Progresion. El tipo Progresion y sus tipos relacionados están definidos en 
Progresiones.hs, el parser está en PrologAHaskell.hs y se llama 
parserProgresion :: Parser Char Progresion. Para saber como usarlo consultad 
pruebaParserProgresion :: String -> IO() en el mismo fichero y aplicaParser 
:: (Parser Char a) -> (String -> a) y leeYParsea :: Parser Char a -> String 
-> IO a del archivo Parsers.hs, si teneis problemas.
