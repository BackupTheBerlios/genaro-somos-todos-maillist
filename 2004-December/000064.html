He pensado que deberias hacer dos parsers uno para lo que yo llamo ritmo 
horizontal y otro para el ritmo vertical. No lo hago por capricho sino 
porque a lo mejor el ritmo horizontal nos vale para alguna otra cosa, ya que 
marca cuando se ejecutan algunas notas del acorde (del AcordeOrdenado). 
Quizas valga para el bajo o la bateria, no lo se. De todas formas en el caso 
de que no valga para nada no hemos perdido nada.
Al grano, el ritmo horizontal es asi:

type Acento = Int
type URH = (Acento, Dur)
type PatronHorizontal = [URH]

PatronHorizontal es lo que debe parsear. EL ritmo vertical es asi:

type Voz = Int
type Ligado = Bool
type URV = (Voz, Ligado)
type PatronVertical = [URV]

Igual que antes PatronVertical es lo que hay que parsear

Y hablando de parsers... ya que vamos a tener unos cuantos parsers 
deberiamos hacer algo sistematico en cuanto a la generacion y la 
nomenglatura. En cuanto a la generacion no se como van pero parece que son 
faciles. En cuanto a la nomenglatura deberiamos llamar al nombre del archivo 
parserLoQueSea.hs y tener un archivo que sea parser.hs que simplemente sea 
un modulo que importe los otros parser
