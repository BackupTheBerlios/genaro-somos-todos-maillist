	Llamadas a Prolog para las progresiones:

	Para hacer una progresión están los siguientes argumentos:

	Directorio_Trabajo: Directorio de trabajo actual

	Instrucción: existen 3, Crea_Progresión, Muta_Progresión

	Ruta_Origen: Ruta de la progresión original

	Ruta_Destino: Ruta destino de la progresión obtenida

	N_Compases: Número de compases utilizados para la progresión a generar

	MUTACIONES: existen 5 tipos de mutaciones,
	  “T1 n” Realiza n mutaciones de tipo 1, es decir, JUNTA ACORDES
	  “T2 n” Realiza n mutaciones de tipo 2, es decir, SEPARA ACORDES
	  “T3 n” Realiza n mutaciones de tipo 3, es decir, CAMBIA ACORDES
	Estos 3 primeros acordes pueden ser agrupados en el tipo A, es decir “TA n”
	  “TA n” Realiza n mutaciones de cualquiera de los 3 primeros tipos

	  “T4 n” Realiza n mutaciones de tipo 4, es decir, DOMINANTE SECUNDARIOS
	  “T5 n” Realiza n mutaciones de tipo 1, es decir, 2ª MENOR SÉPTIMA
	Estos 2 últimos acordes pueden ser agrupados en el tipo B, es decir “TB n”
	  “TB n” Realiza n mutaciones de cualquiera de los 2 últimos tipos

	Y si no se quieren especificar el tipo de mutaciones se diría “MT n” 
realiza n mutaciones.
	*Es obligatorio nombrar los 5 tipos de mutaciones si no se nombra ningún 
tipo que los agrupe, es decir, si decidimos
	hacer T1 a T2 b TB c, deberíamos especificar T3 d, aunque ese d sea 0.
